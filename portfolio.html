眼鏡電商網站切版 01

💡作品簡介 :

這是一個以響應式設計為核心的電商網站切版，針對眼鏡產品展示需求，設計了適配電腦與手機的跨裝置界面，提升用戶體驗感受。

🎯網站特色 :

  1. 為了提供使用者在手機介面的最佳體驗，使用flexbox設計導覽列，讓清單項目換行顯示，頁尾則隱藏冗長資訊。
  2. 響應式更改商品banner的主視覺圖片，在不同介面都能確實傳達商品slogn。
  3. 從購物流程到顧客推薦的區塊具備響應式設計，讓畫面對稱完整呈現頁面訊息給用戶。

❤️成果與反饋：

  1. 目前已具備JavaScript技術，導覽列設計也可使用像履歷樣式，在手機介面下使用展開與摺疊的漢堡選單，讓畫面更簡介。
  2. 漢堡選單的結構寫在nav標籤中讓網頁構更清晰，讓瀏覽器清楚有助於SEO優化。
  3. 接續第2點，漢堡選單結構與樣式開始建立後在大介面時隱藏，在小介面時改為顯示，更好結構樣式與顯示控制分離，重複定義樣式易於維護管理。
※電腦版頁面 | 可點擊圖片進入網頁
※手機版頁面

✔️使用技術：

  • HTML5 與 CSS3框架與樣式建構。
  • 採用RWD自適應設計，適配不同螢幕尺寸
  • GitHub Pages 作品展示。

LOGO設計與數位推廣優化 03

💡作品緣起 :

自家舊式招牌全以文字呈現且沒有精油服務的主題，無法完整傳遞店家服務內容與明顯的特色形象給客戶原招牌設計， 過於單調以純文字為主，缺乏視覺吸引力，且未能有效傳達店家獨特的『髮型設計』與『精油服務』兩大核心特色， 導致品牌形象模糊，無法吸引新客群。

🔍作品發想 :

優化LOGO設計：

●設計概念： 結合剪刀、髮梳、薰衣草視覺元素，象徵剪出亮麗人生。

●色彩運用： 選用溫暖的紫色與橙色，展現專業質感與親和力。

●設計工具： 使用 Illustrator和Photoshop 完成高解析度LOGO設計。

數位品牌推廣 :

●將新LOGO設置為店家LINE群組大頭照，強化視覺辨識度。

●更新Google地圖商家資訊，確保品牌形象一致，吸引更多線上顧客。

品牌故事行銷:

●LINE群中發布LOGO理念與品牌故事，強調「精油護髮」雙重價值。

●分享客戶見證與設計過程，促進客戶對品牌的情感連結。

🎯成品亮點 :

視覺形象提升：

新LOGO設計提升品牌識別度，吸引鄰近新顧客到店詢問。

線上曝光增加：

LINE群組的品牌大頭照更換後，群組互動率頻繁，增加顧客回訪與諮詢

增加品牌影響:

透過LOGO故事行銷，成功吸引客戶對品牌服務認同，提升滿意度。
